My Projects

Four projects I've built while learning to code. Each card links through to its repository on GitHub.

Popcorn-Picks

Popcorn Picks is a movie discovery web app built with PHP, HTML, CSS, JavaScript and a movie database. Users can browse featured titles, search films by keyword, and view detailed movie pages.

PHP

Study-Hive

Study Hive is a student-focused study and booking platform designed to make learning more organised, accessible and engaging. Built with HTML, CSS and JavaScript, it features login and signup pages.

HTML

Colour-Rush

Colour Rush is a reaction-time game built with Python and Pygame. Players must wait for a rectangle to change colour and then press the spacebar as quickly as possible. The game tracks reaction speed.

Python
