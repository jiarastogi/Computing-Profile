My Past Experiences

Over the past few years, I have developed a strong interest in computing through using everyday technology such as phones, computers, games, and the internet, as well as more advanced tools like 3D printers, laser printers, and design software. My three main computing experiences have been:

  1. Web Design
  2. Using Software Applications
  3. Learning Python

I've explored my experiences further below, to provide a deeper understanding of my involvement with these projects and how I've learnt about computing and further enhanced my skills.

Web Design

Through working with my parents in the family business, Get RARE Properties, I have gained practical experience in web design as a Digital Marketing Specialist. Over the past five years, I have helped create multiple web pages and landing pages. I first worked with Wix, which helped me understand the basics of website structure and design, before moving to Ontraport.

Ontraport gave me experience with a more advanced system because it combines web pages with customer relationship management features. This allowed me to understand how websites can connect with forms, messages, contact databases, and business processes. Through this experience, I learned more about web design, SEO, database management, and website security.

The links of the websites are https://getrare.com.au and https://thepropertywealthblueprint.com/.

Using Software Applications

Through school, university, and extracurricular activities, I have learned to use a range of software applications, including Canva, HTML, Visual Studio Code, Lightroom, Descript, and AutoCAD. These tools have helped me develop skills in communication, design, problem-solving, information management, and digital systems.

Using these applications has shown me that computing is not limited to programming. It can also support creativity, organisation, communication, and technical work. This experience has helped me see how computing skills can be useful in many future careers.

Learning Python

As part of my HSC subject Software Engineering, I learned Python through the Grok Academy platform. One of the projects I completed involved creating a reaction speed test game combined with a colour blindness test. This gave me the opportunity to explore what Python can do through a practical and creative task.

This project helped me learn about programming logic, game design, debugging, and error handling. It also increased my confidence in coding and showed me how computing can be used to create interactive tools.
